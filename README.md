# YouTube Thumbnails - Project Showcase

A modern, static landing page to showcase the MLOps pipeline and dataset.

## Setup
1. Open `index.html` in your browser to preview.
2. Edit `style.css` to change colors/theme.

## Deployment
1. Push to GitHub.
2. Go to Repo Settings -> Pages.
3. Select `main` branch as source.
4. Your site will be live at `https://your-username.github.io/youtube-thumbnails-site/`.
